Create a Python script in VSCode using Copilot Inline Chat
In this demonstration, I use GitHub Copilot Inline Chat in VSCode to create a new Python script that: - call the GitHub API to get the top 3 repositories of a user/organization - return the list as text, then refactored to return JSON list
Building function from Spring Boot annotation using Copilot completion in IntelliJ
Creation of a new method through completion, the name, parameters, and code of this function are generated from a simple Spring Boot annotation.
Asking Copilot, in French, how to create and publish container images
This sample show how GitHub Copilot Chat can help a developer to write a workflow.
Moving from Shell to Terraform with GitHub Copilot
In this demonstration I show how to take a Shell script file that contains many call to the Azure CLI tool (az) to create resources and convert this to a set of Terraform files. In addition to the convertion from Shell to Terraform with Azure, I am also asking for suggestion for Terraform file for AWS Deployment.
Converting a Spring Boot Java REST class to Python with Flask with Copilot
In this demonstration I am using GitHub Copilot to propose a new implementation of a Java Spring Boot REST Controller in Python with Flask and use SQLAlchemy for the database access
Finding, Explaining, Testing and Fixing Code Vulnerability with GitHub Copilot
In this demonstration I use GitHub Copilot in IntelliJ to find some vulnerability in a class, this does NOT replace the security control done in your CI/CD platform!) Once the vulnerabilities found, Copilot explain how to fix it, test the fix. This is done in a Java Spring Boot Application (Petclinic) in IntelliJ.
Generate new test in VSCode with Copilot
In this demonstration I use GitHub Copilot to create new test, and add test to my existing test.
Generate Documentation and Comments for Typescript function
In this demonstration I use Copilot to add some JavaScript Documentation for a method, and add comments in the code.
Securing & Refactoring in VSCode with Copilot
In this demonstration, I use Copilot Chat in VS Code to refactor some code: - extract all SQL queries database interactions from an API Controller into a new Repository Class - Rework the SQL Queries when they are not save commenting them with some explanation - Add Try Catch block in the new code for exception management
Create a TypeScript class from functionnal specifications, in French
In this demonstration, I use VSCode and GitHub Copilot to use a specification, written in French, and create a new TypeScript Class that respects the specifications.
